# 期货日历应用 (Futures Calendar) - 设计文档

## 应用概述

期货日历是一个移动应用，帮助用户快速查看每天的期货品种交易信息。用户可以点击日历上的任意日期，应用会在下方以滑动动画显示该日期的期货品种列表。

## 屏幕列表

### 1. 主屏幕 (Home Screen)
- **内容**: 日历组件 + 期货品种列表
- **布局**: 
  - 上方：月份日历（网格布局，7列）
  - 下方：当前选中日期的期货品种列表（可滑动）

### 2. 期货品种详情屏幕 (Futures Detail Screen)
- **内容**: 单个期货品种的详细信息
- **布局**:
  - 品种名称、代码
  - 当前价格、涨跌幅
  - 交易时间
  - 返回按钮

## 主要功能

### 日历交互
- 显示当前月份的日历
- 点击日期后高亮选中状态
- 支持上一月/下一月导航
- 显示当前日期的特殊标记

### 期货品种展示
- 点击日期后，下方列表以向上滑动动画显示该日期的期货品种
- 显示品种名称、代码、价格、涨跌幅
- 支持点击品种查看详情
- 列表支持上下滑动

## 关键用户流

1. **查看今天的期货品种**
   - 用户打开应用 → 日历自动显示当前月份 → 当前日期已高亮 → 下方显示今天的期货品种列表

2. **查看特定日期的期货品种**
   - 用户点击日历上的日期 → 日期被高亮 → 下方列表以滑动动画更新 → 显示该日期的期货品种

3. **查看期货品种详情**
   - 用户在列表中点击品种 → 导航到详情屏幕 → 显示完整信息 → 返回主屏幕

## 数据结构

### 期货品种 (Futures)
```typescript
interface Futures {
  id: string;
  code: string;           // 期货代码，如 "CU2406"
  name: string;           // 品种名称，如 "沪铜"
  price: number;          // 当前价格
  change: number;         // 价格变化
  changePercent: number;  // 涨跌幅 (%)
  tradingHours: string;   // 交易时间，如 "09:00-15:00"
  date: string;           // 日期 (YYYY-MM-DD)
}
```

### 日期期货映射 (DateFuturesMap)
```typescript
interface DateFuturesMap {
  [date: string]: Futures[];  // 日期 -> 期货品种列表
}
```

## 设计原则

1. **移动优先**: 应用针对竖屏(9:16)单手操作设计
2. **简洁易用**: 最小化交互步骤，快速查看信息
3. **视觉反馈**: 点击日期时有明确的高亮和动画反馈
4. **iOS风格**: 遵循 Apple Human Interface Guidelines，使用原生 iOS 设计语言
5. **性能优先**: 日历和列表使用高效的渲染方式，避免不必要的重新渲染
